Wnmp

Welcome to Wnmp!

Wnmp is a client which launches Nginx, MariaDB & PHP on Windows Vista - Windows 8. When Wnmp is distributed it includes Nginx, MariaDB, PHP, & phpMyAdmin.

By default, Wnmp installs to C:\Wnmp.

Configuration notes

Default editor

The default editor for Wnmp configuration and log files is 'notepad'. You can set the Editor setting by going to File->Wnmp Options and clicking .. and selecting the location to the editors exe.

Start Wnmp with Windows

The default option for starting Wnmp with Windows is disabled. You can set the Start Wnmp with Windows setting by going to File->Wnmp Options and checking the check box right next to the Start Wnmp with Windows label.

Launch all applications when Wnmp starts

The default option for starting all applications when Wnmp starts is disabled. You can set the Start all programs on launch setting by going to File->Wnmp Options and checking the check box right next to the Start all programs on launch label.

Minimize Wnmp to tray

The default option for minimizing Wnmp to the tray is disabled. You can set the Minimize Wnmp to tray setting by going to File->Wnmp Options and checking the check box right next to the Minimize Wnmp to tray label.

Automatically check for updates

The default option for automatically checking for updates is enabled and to check for updates every week. You can set the Automatically check for updates setting by going to File->Wnmp Options and checking the check box right next to the Automatically check for updates label and selecting the desired frequency to check for updates from the combo box.

Reporting problems

Before you report any problems, please consult the Wnmp FAQ Page and see if your question is already in our list of Frequently Answered Questions (FAQ).

If you cannot find an answer to your question, please feel free to send mail to the Wnmp mailing list, at wnmp-users@getwnmp.org. Remember, the more useful information you include in your report, the easier it will be for us to help you!
